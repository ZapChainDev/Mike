# Train with Mike - Next.js Landing Page

A high-energy fitness landing page built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Hero Section**: Full-width background with dramatic gym imagery
- **Responsive Header**: Navigation with links and member login
- **Social Media Integration**: YouTube, Instagram, and X (Twitter) icons
- **Chat Widget**: Fixed bottom-right chat button
- **Modern Stack**: Next.js 14 App Router, TypeScript, Tailwind CSS
- **Lavender Purple Accent**: #A376FF throughout the design

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- React Icons

## Build

To create a production build:

```bash
npm run build
npm start
```
